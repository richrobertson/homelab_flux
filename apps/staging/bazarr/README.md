# Bazarr (Staging)

Staging overlay for Bazarr.

## Purpose

- Validates subtitle automation updates and dependencies in staging.
- Applies environment-specific settings on top of the Bazarr base.

## In this folder

- Overlay patches and staging kustomization entries.

## Typical staging deltas

- Test hostnames, lower resource requests, and staging secret refs.
