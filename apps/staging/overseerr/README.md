# Seerr (Staging)

Staging overlay for Seerr.

## Purpose

- Validates request-workflow changes before promotion.
- Applies environment-specific differences on top of base manifests.

## In this folder

- Test endpoints, staging credentials, and reduced replica/resource settings.
