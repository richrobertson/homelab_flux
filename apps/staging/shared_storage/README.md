# Shared Storage (Staging)

Staging overlay for shared storage resources.

## Purpose

- Tests storage definition changes before production adoption.
- Applies staging capacity/class differences over base shared storage manifests.

## In this folder

- Staging patch files and overlay composition resources.

## Typical staging deltas

- Smaller capacities, test storage classes, and non-prod PVC bindings.
