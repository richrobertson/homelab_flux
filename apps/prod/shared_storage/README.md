# Shared Storage (Prod)

Production overlay for shared storage resources.

## Purpose

- Provides storage definitions used by multiple production workloads.
- Applies production capacity and class choices over the base storage layer.

## In this folder

- Production storage patch manifests and overlay wiring.

## Typical production deltas

- Larger capacities, production storage classes, and durable binding choices.
