# Prowlarr (Prod)

Production overlay for Prowlarr.

## Purpose

- Maintains live indexer management and search integrations.
- Applies production credentials and endpoint values over base config.

## In this folder

- Production patch manifests and kustomization references.

## Typical production deltas

- Live indexer/API credentials, stable connectivity settings, and production sizing.
