# Sonarr (Prod)

Production overlay for Sonarr.

## Purpose

- Runs live TV automation services.
- Applies production route, secret, and scaling values over base Sonarr config.

## In this folder

- Production patch manifests and overlay composition resources.

## Typical production deltas

- Stable ingress domains, production credentials, and tuned resource allocations.
