# Mealie (Prod)

Production overlay for Mealie.

## Purpose

- Runs live recipe management services with production settings.
- Applies environment-specific values over the base Mealie manifests.

## In this folder

- Production overlay patches and kustomization resources.

## Typical production deltas

- Public hostnames, production secrets, and durable persistence/resource settings.
