# Syncthing (Base)

Base manifests for Syncthing file synchronization services.

## Purpose

- Defines shared chart source and baseline app values.
- Establishes common sync service defaults and resource patterns.
- Acts as reusable configuration for all environment overlays.

## In this folder

- Helm source/release resources and any shared support manifests.

## Overlay expectations

- Node placement, ingress exposure, and environment secrets are overlay concerns.
