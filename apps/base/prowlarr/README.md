# Prowlarr (Base)

Base manifests for Prowlarr indexer aggregation and management.

## Purpose

- Captures common deployment source and release defaults.
- Standardizes base connectivity assumptions for media automation apps.
- Serves as the shared config layer for both environments.

## In this folder

- Source/release manifests and supporting common resources.

## Overlay expectations

- Provider credentials and environment-specific integrations are defined in overlays.
