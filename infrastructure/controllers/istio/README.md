# Istio

Controller resources for service mesh control-plane components.

## Purpose

- Manages Istio control-plane installation and lifecycle.
- Provides mesh APIs/features used by gateway and app routing.

## In this folder

- Includes `gateway-api/` resources for Gateway API definitions.
